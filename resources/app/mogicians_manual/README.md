## 功能
【说】：语录、采访、访问对话、诗词、论文等作品。

【学】：考据、同人、段子、知乎精选等知识。

【逗】：高清、原生、颜艺、成语、鬼畜、恶搞、暴走漫画等表情，支持动图。

【唱】：古典、流行、动画等音乐。

【赏】：蛤三篇总集、膜学选修教材 等视频。
